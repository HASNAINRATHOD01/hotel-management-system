# Hotel Management System

## Project Overview
Hotel Management System is a role-based web application developed to manage hotel operations such as room booking, room management, and user administration.

The system supports three roles:
- Customer
- Manager
- Admin

---

## Technology Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: MySQL (phpMyAdmin)

---

## User Roles & Features

### Customer
- View available rooms
- Book rooms
- View booking history

### Manager
- Manage room status
- Approve or cancel bookings
- View check-in and check-out details

### Admin
- Manage users and roles
- Manage rooms and pricing
- View reports and revenue details
